bi-his(bimodally interpreted-homoiconic instruction set) current version: 1.0.0(look ma, no head!)

an esoteric tape-based Turing-tarpit with only 5 set of commands, yet with homoiconicity, a module linker system, magic tape cells with complete and direct access to Python and Hy ecosystem(because it's written in Hy) which lives between the low-level memory manipulation of Brainfuck(the syntax is eerily similar) and the high-level homoiconic Lisp evaluation of Hy, which gives you the perfect minimalist experience(or nightmare) of creating almost anything possible in programming, because its Turing complete,but so is Brainfuck but here you have even more usability and support.
